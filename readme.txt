程序使用方式：
	请将本压缩包解压后放置在D盘根目录下;
	将此文件夹改名为mjw_mzy; 		(名称为成员的名字字母缩写)
	然后双击Compiler.exe即可运行编译程序。



各文件夹功能：

Demo文件夹放置的为测试用例，其中正确实例文件夹中放置的均为分析正确的源语言程序，
	错误实例文件夹中放置的均为有错误的源语言程序。
	选择测试文件请选择此文件夹下的测试文件。

Program文件夹下放置有三个源程序代码文件及其执行文件，包括WordAnalyser,GrammarAnalyser,Explainer.此为后端核心代码文件
	此外还有名为Qt_Project的压缩包，为Qt的工程文件，若无特殊要求无需解压。
	UI_Setting文件夹为UI界面配套文件夹，主要放置界面中用到的图像文件。

Result文件夹中存放各程序运行保存的结果，有Word_Result.txt,Grammar_Result.txt,Run_Result.txt三个。

报告及验收PPT文件夹存放有小组成员的电子版报告以及验收的PPT。

Compiler.exe为打包好的可运行文件，只要双击图标即可运行。

readme.txt 即为本文件，介绍程序使用方式以及文件夹构造。
